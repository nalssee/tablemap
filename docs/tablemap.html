tablemap.util.chunk(rs, n, column=None)¶

Returns a list of rows in chunks

Parameters

  • rs – a list of rows

  • n –

    • int => returns 3 rows about the same size

    • list of ints, [0.3, 0.4, 0.3] => returns 3 rows of 30%, 40$, 30%

    • list of nums, [100, 500, 100] => returns 4 rows with break points 100, 500, 1000, but you must pass the column name for the break points like

    chunk(rs, [100, 500, 100], ‘col’)

  • column – column name for break points

Returns

a list of rows
